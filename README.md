# Facial-Beauty-Scoring-Model
The model follows the multi-task learning approach established in the referenced literature. personally I got to know it from "CFO facial beauty and bank loan contracting" by Hrazdil, Li, Lobo, and Zhang (2024). 
Utilizing a ComboNet architecture built upon an SE-ResNeXt-50 backbone. This backbone integrates Squeeze-and-Excitation (SE) blocks, which function as an adaptive attention mechanism to weight the importance of different feature channels. By squeezing spatial information and exciting relevant channels, the network prioritizes informative facial features while suppressing irrelevant background noise. The ResNeXt component complements this by using parallel convolutional paths, allowing the model to learn a more diverse set of facial representations without increasing computational complexity.
To ensure more robust predictions, the architecture diverges into two simultaneous output heads: a regression branch for continuous scoring and a classification branch for discrete levels. This multi-task strategy acts as a regularizer during the learning process, forcing the feature extractor to capture both broad aesthetic categories and fine-grained variations. The model was trained on the SCUT-FBP5500 dataset, the academic standard for this task, which contains 5,500 portraits rated by 60 individual volunteers to establish a highly reliable average ground truth.
The implementation includes a specialized weight-mapping logic to ensure that the parameters learned during the original training phase are correctly translated into the specific model hierarchy used by the pytorchcv library. To prepare the input images, a preprocessing stage uses Dlib’s 68-point landmark detector to perform an affine transformation. This aligns the face to a neutral, centered position, removing environmental variance such as head tilt or camera distance. The resulting output is then clamped and scaled to provide a consistent assessment on a 1–5 scale for the analysis.
